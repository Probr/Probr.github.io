probr-core

setup using python and pip

Install Python 2.7 and PIP. We highly recommended to use Virtualenv to manage the python environment for probr-core. After cloning the project, create a virtual environment for probr outside the probr-core directory: virtualenv .env_probr

Activate the virtualenv: source .env_probr/bin/activate. After this, use pip install -r requirements.txt in the projects directory and install all required environments.

Now use bower install to install the dependencies for the web frontend. All that is left to do now, is to create the initial databases for probr. Use python manage.py migrate. You will be asked to set your password upon your first login.

setup using docker

Install Docker. Setup all necessary services using sh install/docker/setup.sh. As soon as the command has finished, use sh install/docker/deploy.sh.
